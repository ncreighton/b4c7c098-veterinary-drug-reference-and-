# Veterinary Drug Reference and Prescription Management Discord Bot

A Discord bot for veterinary teams providing instant drug reference lookups, dosing calculators, and controlled substance log management. Calculates weight-based dosing for 500 plus veterinary medications, flags dangerous drug interactions, tracks DEA controlled substance inventory with dispense logging, sends prescription refill reminders to clients, and posts daily pharmacy report summaries to staff channels. Built for veterinary hospitals, emergency clinics, and specialty practices.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
